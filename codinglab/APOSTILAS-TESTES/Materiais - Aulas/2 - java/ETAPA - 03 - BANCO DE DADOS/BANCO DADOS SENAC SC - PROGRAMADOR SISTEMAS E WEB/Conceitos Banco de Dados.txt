Existem 2 tipos basicos de Banco de dados
-> Relacional 
-> NoSql

Relacional, relacionado diretamente a tabelas, tendo varias colunas para cada uma delas

NoSQL, relacionado a objetos, e e composto por atributos do tipo chave->valor
